Analyzing NYC-Crime

Project Details: We are gonna use data from NYC OpenData to analyze NYC Crime issues.

Data can be downloaded from https://data.cityofnewyork.us/Public-Safety/NYPD-Complaint-Data-Historic/qgea-i56i

How to run the programs:
For all the .py files, command line: spark-submit --packages com.databricks:spark-csv_2.10:1.2.0 SCRIPT.py
For all the .sh files, command line: sh script.sh
